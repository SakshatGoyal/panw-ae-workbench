# Tooltip

[Source code](packages/tooltip/src/Tooltip/Tooltip.tsx)

## Overview

Glanceable label primitive — one or two lines of text on a dark inverse
ground, anchored to a trigger by a small arrow. Stage rules:

- **surface** — `--ds-surface-inverse-rest`
- **text** — `--ds-text-inverse-rest`
- **radius** — `--ds-radius-tight` (4px)
- **padding** — `8px 12px` (the static-label rule)

Need a heading, image, or paginated stepper? That's a popover, not a
tooltip — import from `@ds/popover` instead. The descriptive variant
that used to live here as `kind="descriptive"` was split into the
popover package; consumers importing `DescriptiveTooltip` should switch
to `Popover`.

This component renders the popup itself; positioning relative to a
trigger is the consumer's responsibility.
