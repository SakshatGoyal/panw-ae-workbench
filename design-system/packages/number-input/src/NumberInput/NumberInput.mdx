# NumberInput

[Source code](packages/number-input/src/NumberInput/NumberInput.tsx)

## Overview

Numeric stepper with text entry plus increment/decrement buttons. Two
background tones, controlled via `value`/`onChange`.

## Composition

The two stepper buttons are `IconButton` from `@ds/button` (imported, not
redefined).

## Accessibility

- The text input is a real `<input type="number">` and inherits its
  semantics. `aria-label` is set to the title when title is a string.
- Stepper buttons are real `IconButton`s with `aria-label="Decrease value"`
  / `"Increase value"`.
