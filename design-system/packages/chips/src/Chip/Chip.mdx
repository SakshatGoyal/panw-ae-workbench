# Chip

[Source code](packages/chips/src/Chip/Chip.tsx)

## Overview

A pill-shaped interactive element used for filters, selections, tokenized
inputs, and tags-with-actions. Supports a leading icon **or** image (mutually
exclusive), an optional close button, an optional dropdown chevron, and an
active/selected state. Dark theme variants are included.

## Accessibility

- The chip body has `role="button"`, `tabIndex={0}`, and supports Enter/Space
  activation when `onClick` is supplied.
- `active` is reflected as `aria-pressed="true"` (toggle-button semantics).
- `disabled` sets `aria-disabled` and removes the chip from the tab order.
- The close button is a real `<button>` with an `aria-label` describing the
  removal target ("Remove \{label\}").
- Leading icon and dropdown chevron are `aria-hidden="true"` (decorative).

## API Change: `renderLeadingIcon` / `renderCloseIcon`

PANW's source rendered fixed inline SVGs for the checkmark and close. The port
exposes `renderLeadingIcon` / `renderCloseIcon` as `ElementType` overrides
defaulting to lucide-react `Check` / `XCircle`.
