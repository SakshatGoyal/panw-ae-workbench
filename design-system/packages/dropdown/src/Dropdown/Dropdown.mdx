# Dropdown

[Source code](packages/dropdown/src/Dropdown/Dropdown.tsx)

## Overview

Single-select rounded dropdown. Two background tones, two heights, with
title and description slots.

## API change: split from PANW `DropdownRounded`

PANW's `DropdownRounded` accepted a `multiselect` flag and rendered chips
inline when multiple values were selected. This port splits that surface
into two components, matching Carbon:

- `@ds/dropdown` (`Dropdown`) — single-select (this package)
- `@ds/multi-select` (`MultiSelect`) — multi-select with chips

Use `Dropdown` when only one value can be chosen.

## Accessibility

- The trigger has `role="combobox"`, `aria-expanded`, `aria-haspopup="listbox"`.
- Menu uses `role="listbox"`, items `role="option"` with `aria-selected`.
- `Enter` / `Space` toggles open/close from the trigger.
- Click outside closes the menu.
