# RadioButton

[Source code](packages/radio-button/src/RadioButton/RadioButton.tsx)

## Overview

A controlled radio input rendered with a custom 16×16 SVG ring + dot. Wraps a
visually-hidden native `<input type="radio">` so keyboard, screen reader, and
form-submission semantics behave like a standard radio.

## Accessibility

- Uses a real `<input type="radio">` (visually hidden via clip rect) so native
  keyboard activation, focus, and screen reader semantics all work.
- The custom SVG icon is `aria-hidden="true"` — assistive tech reads the input.
- The wrapping `<label>` makes the entire row clickable and associates the
  label text with the input.
- `disabled` sets the native `disabled` attribute, which removes the input
  from the tab order.

## API Change: `onChange` signature

PANW's source emitted `onChange(checked: boolean)` — a single boolean. The
port emits `onChange(checked, event)` so consumers have access to the native
`ChangeEvent` (Carbon convention).
