# MultiSelect

[Source code](packages/multi-select/src/MultiSelect/MultiSelect.tsx)

## Overview

Multi-select rounded dropdown. Selected values render as chip-tags; values
beyond `visibleChipCount` collapse into a `+N` badge.

## API change: split from PANW `DropdownRounded`

PANW's unified `DropdownRounded` is split into `@ds/dropdown` (single
select) and `@ds/multi-select` (this component). Use this when more than
one value can be chosen at once.

## Composition

The chips are rendered with `@ds/tags` `Tags` (imported, not redefined).
