# Search

[Source code](packages/search/src/Search/Search.tsx)

## Overview

A text input with leading search glyph and an inline clear button. Three
sizes, three corner shapes (standard / rounded / pill), and two background
tones.

## Composition

The clear button is the `IconButton` from `@ds/button` (imported, not
redefined).

## Accessibility

- The wrapper has `role="search"`.
- Clear button is a real `IconButton` with `aria-label="Clear search"`,
  `tabIndex={-1}` so it does not steal focus from the input.
- `aria-label` on the input falls through from `placeholder`.

## API Change: `searchIcon` (ReactNode) → `renderSearchIcon` (ElementType)

PANW accepted a pre-rendered `searchIcon: ReactNode`. This port follows
Carbon's convention — pass a component:
