# Toggle

[Source code](packages/toggle/src/Toggle/Toggle.tsx)

## Overview

A switch with an optional label, info icon, and status text (Yes/No). The
underlying input is a real `<input type="checkbox" role="switch">` so keyboard
and screen reader behavior match a native switch.

## Accessibility

- Renders a real `<input type="checkbox" role="switch">` so AT announces it
  as a switch and native keyboard activation works.
- The `<label>` wraps the input, making the entire row clickable.
- The info icon is `aria-hidden="true"` — provide additional context via the
  label or a `Tooltip` if you need it announced.
- Disabled uses the native `disabled` attribute.

## API Change

PANW's source emitted `onChange(on: boolean)`. The port emits
`onChange(on, event)` so consumers can access the native event.
