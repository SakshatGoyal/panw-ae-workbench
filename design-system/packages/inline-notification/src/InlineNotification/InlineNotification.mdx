# InlineNotification

[Source code](packages/inline-notification/src/InlineNotification/InlineNotification.tsx)

## Overview

A horizontal banner with severity icon, message, and optional close
button. Four severity types and two prominence levels.

## Composition

Close button is the `IconButton` from `@ds/button` (imported, not redefined).

```jsx

<InlineNotification type="info" infoType="product-level">High contrast</InlineNotification>
```

## Accessibility

- The root has `role="status"` for screen-reader announcements.
- The close button is a real `IconButton` with `aria-label="Close notification"`.
