## Overview

Tags are pill-shaped labels used to categorize, filter, or mark content. The
PANW Tags component supports 17 colors across two contrast levels, two sizes,
an optional leading icon, and an optional close (dismiss) button.

## Colors

17 options are available: 16 categorical colors — `grey`, `accent`, `red`,
`green`, `orange`, `slate`, `lavender`, `purple`, `pink`, `yellow`,
`bronze`, `olive`, `lime`, `jade`, `teal`, `cobalt` — plus the structural
`neutral` variant. See the AllVariants story for a visual matrix.

```jsx
<Tags label="Accent" color="accent" />
<Tags label="Red" color="red" />
<Tags label="Cobalt" color="cobalt" />
```

## Contrast

`low` (default) uses a pastel background with dark text. `high` uses a
saturated background with light text.

<Canvas of={TagsStories.HighContrast} />

```jsx
<Tags label="Low" color="accent" contrast="low" />
<Tags label="High" color="accent" contrast="high" />
```

## Sizes

Two sizes: `default` (12px text, 4×8 padding) and `large` (14px text, 8×12
padding).

```jsx
<Tags label="Default" size="default" />
<Tags label="Large" size="large" />
```

## With Icon

Pass a lucide-react (or any React) component to `renderIcon` and set
`icon=\{true\}`. Defaults to `Sun` (matches PANW's default "light" SVG).

<Canvas of={TagsStories.WithIcon} />

```jsx
import { Star } from 'lucide-react';

<Tags label="Featured" icon renderIcon={Star} />
```

## Dismissable

Set `close=\{true\}` and pass an `onClose` handler. The close button receives an
auto-generated `aria-label` of `Remove \{label\}`.

<Canvas of={TagsStories.Dismissable} />

```jsx
<Tags label="Dismissable" close onClose={() => removeTag()} />
```

## Accessibility

- The leading icon is `aria-hidden="true"` — it is decorative.
- The close button is a real `<button>` with an `aria-label` describing the
  removal action ("Remove \{label\}").
- The close button has a visible `:focus-visible` outline.
- Tags are non-interactive by default (`cursor: default`, `user-select: none`).

## API Change: `renderIcon`

**Deliberate change from PANW `Tags`:**

PANW's source accepted `customIcon` and `customCloseIcon` as `ReactNode` — you
passed already-rendered elements:

```jsx
// PANW (old)
<Tags label="Foo" icon customIcon={<MyIcon />} />
```

The ported `Tags` uses Carbon's `renderIcon: ElementType` convention — pass the
component, not a rendered element:

```jsx
// Ported (Carbon convention)
import { Star } from 'lucide-react';

<Tags label="Foo" icon renderIcon={Star} />
```

This lets the component control sizing (12px / 16px depending on `size`) and
inject `aria-hidden` correctly.
