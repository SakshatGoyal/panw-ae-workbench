# Tooltip

[Source code](packages/tooltip/src/Tooltip/Tooltip.tsx)

## Overview

Two variants:

- **default** — dark background with a single line of text. Use for short
  contextual hints.
- **descriptive** — light card with heading, image slot, body text, and a
  page stepper. Use for guided onboarding-style hints. The descriptive
  variant's stepper navigation is built with the `@ds/button` `IconButton`
  (imported, not redefined).

This component renders the popup itself; positioning relative to a
trigger is the consumer's responsibility (compose with your own anchor
strategy or wrap in a future `Popover` primitive).

## Accessibility

- Root has `role="tooltip"`.
- Decorative arrow uses `aria-hidden="true"`.
- Stepper buttons are real `IconButton`s with `aria-label="Previous"` and
  `"Next"`. They become `disabled` at the boundaries.
